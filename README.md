# 🎫 Cupony - Coupon Aggregation Web App

**Cupony** is a full-stack web application that aggregates and displays discount codes from various retailers.  
Users can easily browse active coupons, while administrators have access to a dedicated panel to manage the offers.  
The project was built collaboratively by a development team as part of a professional training program, under the mentorship of industry experts.

## ✨ Features

- 🛍️ Browse and use active discount coupons  
- 🔐 User authentication (login/register)  
- 🧑‍💼 Admin panel for managing coupons  
- 🔎 Real-time search functionality  
- 💡 Simple and responsive UI with modern design

## 🛠️ Technologies Used

**Frontend**

**Backend**

## 🚀 Getting Started

* If you don't have `pnpm` in your global path please install
  * `npm install pnpm -g`
* Navigate to the project path
* install all monorepo packages
  * `pnpm install`


### Local Server Commands
* For running dev client - `pnpm dev:client`
* For running dev all app  - `pnpm dev`
